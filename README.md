# Popup Campaign Funnel Analyzer

A small, clean Vue.js web application designed to help marketers understand step-level funnel performance for their popup campaigns. It visually represents the conversion funnel, automatically highlights the step with the biggest drop-off, and provides simple, actionable insights based on the data.

## Prerequisites

- **Node.js**: v24.14.0 (or newer recommended)
- **npm**: Included with Node.js

## Getting Started

To run this application locally, follow these simple steps in your terminal:

1. **Install the dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

## View the App

Once the development server is up and running, open your web browser and navigate to the following URL:

👉 **[http://localhost:5173/](http://localhost:5173/)**

*(Note: If port 5173 is already in use, Vite will automatically assign the next available port, which will be displayed in your terminal.)*
